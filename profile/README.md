## Fintrack

Сервис финансовой грамотности, позволяющий вести учет финансов и инвестиций. Аналогичные сервисы зачастую требуют покупать подписку. Помимо этого, хранение данных о своих финансах на серверах третьих лиц может быть опасно, в то время как open source self-hosted решение вызывает больше доверия

## Реализованная функциональность

Банковские фичи:
* Можно вносить доходы и расходы руками
* Статистика доходов - расходов (по категориям, по датам и тп)

Инвестиционное:
* Текущий баланс брокерского счета
* Изменение баланса или стоимости бумаги за промежуток времени
* Календарь выплат по облигациям и дивидендам
* Расчет денег, которые поступят в течение года
