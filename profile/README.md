## Fintrack

Сервис финансовой грамотности, позволяющий вести учет финансов и инвестиций. Аналогичные сервисы зачастую требуют покупать подписку. Помимо этого, хранение данных о своих финансах на серверах третьих лиц может быть опасно, в то время как open source self-hosted решение вызывает больше доверия

## Реализованная функциональность

Банковские фичи:
* Можно вносить доходы и расходы руками
* Статистика доходов - расходов (по категориям, по датам и тп)

Инвестиционные фичи:
* Текущий баланс брокерского счета
* Изменение баланса или стоимости бумаги за промежуток времени
* Календарь выплат по облигациям и дивидендам
* Нотификации

## Описание репозиториев

* `frontend` - веб-приложение, написанное на Flutter
* `backend-common` - репозиторий, содержащий контракты микросервисов
* `backend-master` - мастер-сервис, аггрегирующий в себя данные с других сервисов и предоставляющий API для фронтенда
* `analyzer` - сервис для расчёта статистики
* `market-wallet` - сервис для интеграции API банков и брокеров
* `notification` - сервис нотификаций
* `deploy` - репозиторий для `docker-compose.yml` файла, который запускает все сервисы

## Деплой

* Пройтись по всем репозиториям (кроме `backend-common` и `deploy`) и выполнить сборку и загрузку Docker-образа, следуя инструкциям в этих репозиториях
* На целевой машине склонировать `deploy`-репозиторий и выполнить команду `docker compose up`
